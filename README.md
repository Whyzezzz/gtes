# 🎮 GameStore

A full-stack online game store developed with **Laravel**, **PHP**, **MySQL**, and **JavaScript**. The application allows users to browse games, purchase them securely using Stripe, download purchased content, and manage their accounts through a modern and responsive interface.

---

## 📖 Overview

GameStore was developed as my final project for my vocational course in Computer Science. The goal was to create a complete e-commerce platform for digital games, implementing real-world features such as online payments, authentication, email notifications, API integration, and an administration panel.

---

## ✨ Features

### 👤 User

- User Registration
- Email Verification
- Secure Login
- Password Recovery
- Browse Game Library
- Search Games
- Game Details
- Shopping Cart
- Stripe Payment Integration
- Purchase History
- Download Purchased Content
- Customer Support
- Purchase Confirmation Emails

- Dashboard
- Manage Users
- Manage Games
- Manage Purchases
- Manage Customer Support
- View Platform Statistics

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Whyezzzz/gamestore-laravel.git
```

Install dependencies

```bash
composer install
```

Create the environment file

```bash
cp .env.example .env
```

Generate the application key

```bash
php artisan key:generate
```

Run the migrations

```bash
php artisan migrate
```

Start the server

```bash
php artisan serve
```
